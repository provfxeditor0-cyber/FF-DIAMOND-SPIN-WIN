GitHub Setup Guide

  • 1 Visit Site Go to github.com/signup in your browser.
  • 2 Details Enter your Email, Create Password, and Choose Username.
  • 3 Verify Complete the puzzle/captcha and enter the Launch Code sent to your email.
  • 4 Create Repository Click the + icon in top right and select "New repository". Name it (e.g., "my-app").
  • 5 Upload Files Open your new repository. Click Add file > Upload files.
  • 6 Drag & Drop Drag your website file (index.html) and drop it into the box.
  • 7 Commit Changes Scroll down and click the green "Commit changes" button to save.
  • 8 Finish Copy your repository link. It is now ready for Adsterra!
